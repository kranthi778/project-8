# Malware Threats and Analysis

This project helps us learn about malware. It is based on **CEH v13 – Module 07**. The project shows how to understand malware analyze it statically and check VirusTotal reports. We also learn to monitor network connections and processes in a cybersecurity lab.

The project is about how security analysts check files and monitor system activity. They use tools that're well-known in the industry.

---

# Project Objectives

- Learn the basics of malware.

- Know common types of malware and threats.

- Analyze malware statically using VirusTotal.

- Understand VirusTotal reports.

- Learn about dynamic malware analysis.

- Watch active network connections using TCPView.

- Check network connections using CurrPorts.

- Investigate running processes using Process Explorer.

# Project Workflow

## Part 1 – Introduction to Malware Threats

- Learn about malware.

- Know malware categories.

- Understand cyber threats.

## Part 2 – Static Malware Analysis using VirusTotal

- Upload a sample to VirusTotal.

- Check detection ratio.

- Analyze antivirus reports.

- Get file information.

## Part 3 – Interpreting the VirusTotal Report

- Analyze detection results.

- Check file properties.

- Understand hash values.

- See vendor detection results.

## Part 4 – Understanding Static and Dynamic Malware Analysis

- Learn analysis.

- Learn analysis.

- Compare analysis.

## Part 5 – Monitoring Network Connections with TCPView

- Watch TCP connections.

- Watch active UDP connections.

- Identify processes.

- Find suspicious network activity.

## Part 6 – Monitoring Network Connections with CurrPorts

- Check ports.

- Watch active connections.

- Get process information.

- Monitor connections.

- process hierarchy.

- Check CPU usage.

- Check memory usage.

- Investigate processes.
